试题编辑
-------------

编辑试题须知
````````````

- 可在试卷编辑中直接添加试题，或在题库中编辑试题后导入试卷；

- 所有类型的试题均支持上传音频和图片；

- 其中简答、填空、录音题可不设置正确答案，需要开启人工判分；若设置固定正确答案则跟答案完全一样才得分；

- 判断、单选、多选、拖拽题、完形填空题必须设置正确答案；其中拖拽题为拖拽设置正确选项；

- 复合题增加下设小题可以为：单选、多选、填空、简答。

- 填空题的填空项不可随意添加到文字中间，需编辑一部分添加一个空。
